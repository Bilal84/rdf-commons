-----------------------------
Triple Converter Command Line
-----------------------------

bin$ ./tripleconverter.sh

usage: org.sindice.rdfcommons.tripleconverter.TripleConverter OPTIONS <inputfile>
Supported options are:
 -bu,--base-uri <arg>        The base uri: default 'http://sindice.com/base-uri'
 -if,--input-format          Supported formats are: [RFXML, NTriples, Trig, Turtle, NQuads]
 -O,--output-file <arg>      The output file
 -of,--output-format <arg>   Supported formats are: [RFXML, NTriples, Trig, Turtle, NQuads]

This utility tool helps performing conversions across triple representation formats.
